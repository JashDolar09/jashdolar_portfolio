## ✨ About the Project
- A sleek, modern, and highly responsive portfolio website  
- Designed to showcase my journey as a Web Developer  
- Built with performance, clean UI, and user experience in mind  
- Focused on interactive and engaging design  

---

## 🛠️ Tech Stack
This project uses a modern development stack:

- **Framework:** Next.js 15 (App Router)  
- **Styling:** Tailwind CSS  
- **Components:** Shadcn UI  
- **Icons:** Lucide React  
- **Animations:** Custom scroll-reveal & CSS transitions  
- **Deployment:** Netlify / Vercel

### 🚀 Getting Started

To run this project locally:

- Clone the repository:
```bash
git clone https://github.com/JashDolar09/jashdolar_portfolio.git

npm install

npm run dev

```
